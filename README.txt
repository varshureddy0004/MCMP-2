If you are using MobaXTerm or VS Code, you can upload these files easily.

If you are not, you can copy the file's link, and do the following in the terminal:

wget <file link> 

ENSURE YOU CHANGE dl=0 at the end to dl=1.

e.g. if you want to download input_5.dat, do...

wget https://www.dropbox.com/scl/fi/4vliitcmf9kyq6m66s098/input_5.dat?rlkey=ouedjkf032af9h6jwsdjyz6q1&dl=1
